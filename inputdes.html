Input Description

ข้อมูลการสั่งอาหาร

ข้อมูลการสั่งอาหาร = ร้านอาหาร + อาหาร + จำนวน + ที่อยู่จัดส่ง + หมายเหตุเพิ่มเติม

รายละเอียดข้อมูล

ร้านอาหาร: {รหัสร้านอาหาร + ชื่อร้านอาหาร}

  • รหัสร้านอาหาร: รหัสประจำร้านอาหาร (เช่น 1, 2, 3 เป็นต้น)
  • ชื่อร้านอาหาร: ชื่อร้านอาหารที่เลือก

เมนู: {รหัสอาหาร + ชื่ออาหาร}

  • รหัสอาหาร: รหัสประจำอาหาร (เช่น 1, 2, 3 เป็นต้น)
  • ชื่ออาหาร: ชื่ออาหารที่เลือก

จำนวน: {จำนวนอาหารที่ต้องการสั่ง} (จำนวนเต็ม ≥ 1)

ที่อยู่จัดส่ง: {เลขที่บ้าน + ถนน + ตำบล + อำเภอ + จังหวัด + รหัสไปรษณีย์}

  • เลขที่บ้าน: หมายเลขบ้านที่จัดส่ง
  • ถนน: ชื่อถนนที่อยู่จัดส่ง
  • ตำบล: ชื่อตำบลที่อยู่จัดส่ง
  • อำเภอ: ชื่ออำเภอที่อยู่จัดส่ง
  • จังหวัด: ชื่อจังหวัดที่อยู่จัดส่ง
  • รหัสไปรษณีย์: รหัสไปรษณีย์ของที่อยู่จัดส่ง

หมายเหตุเพิ่มเติม: {ข้อความหมายเหตุเพิ่มเติม} (ไม่จำเป็นต้องกรอก)
